# Service

注册
1. 手机号校验：格式（app和服务端双校验）、未注册
2. 唯一id生成 
3. 加密密码，并储存
4. 设备注册限制 每天每设备注册x个
5. 记录注册日志

登录
1. 输入唯一id或手机号
2. 验证密码
3. 生成jwt token
4. 连续失败x次，限制登录x分钟
5. 记录登录日志

修改资料
1. 输入唯一id
2. 传入要修改的字段
3. 返回修改了的字段
4. 记录日志

修改唯一id
1. 用户注册的时候，会分配一个
2. 每天只能修改一次
3. 验证修改后 合法 和有无重复的

获取用户信息
1. 直接获取

修改密码
1. 查询在x分钟内是否改过，及是否合法
2. 对新密码进行加密，然后改动
